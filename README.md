# 🧮 Calculadora de IMC Interativa

Este aplicativo permite que os usuários insiram seu **peso** e **altura**, e com base nesses dados, calcula o **Índice de Massa Corporal (IMC)**.
Após o cálculo, o app exibe uma mensagem indicando a classificação do usuário: abaixo do peso, peso normal, sobrepeso ou obesidade.

## ✨ Funcionalidades

- Entrada de dados: peso (kg) e altura (cm)
- Cálculo automático do IMC
- Exibição da classificação de saúde:
  - Abaixo do peso
  - Peso normal
  - Sobrepeso
  - Obesidade
- Interface simples e responsiva

## 🛠️ Tecnologias Utilizadas

- React Native
- JavaScript (ES6+)
- Componentes funcionais com hooks

## 📦 Estrutura do Projeto

- `components/` – Componentes reutilizáveis
- `screens/` – Tela principal com formulário e resultado
- `utils/` – Função de cálculo e classificação do IMC
